Internal Hyperlinks

When you click on an internal hyperlink, you will stay on the current page, but will be navigated to a particular HTML element on a different part of the page. This type of anchor is often used for something like a Table of Contents, that enables a website visit to jump down to the part of the page they care about.

If you want to use anchor elements to create an internal hyperlink, you'll need to do two things:

  1. On the element you want to link to, set up an id attribute. (For example: id="where-i-want-to-jump-to")
  2. On the anchor you want to link from, grab the id attribute value you just set up, set that in the anchor's href attribute, and put a # in front of it. (For example: href="#where-i-want-to-jump-to")

Structuring a Table of Contents

As a side note, if you have several anchors in a row, in something like a Table of Contents or some form of navigation, we usually don't want them all lined up in a row, as they would naturally, since they're inline elements. Instead, we probably want them separated in an orderly way, likely on separate lines.

Because of this, we often put anchor elements inside of other elements for better spacing and layouts, such as the Unordered List element and List Item elements we used in the example above.

Icon Library Example (Font Awesome)

Icon libraries are collections of prepackaged code that you can include on your page, so that you can use icons that someone else has already created.

The icon library in this example come from a site called Font Awesome .

Instructions to Use

To use the Font Awesome Library on your page, simply copy the code below in its entirety and paste it in your <head> element, right below your <title> element (and for clean code, split up the attributes across multiple lines; you can't help the length of the attributes going over 80 characters, but you can at least make them easier to read).

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.2.0/css/all.min.css" integrity="sha512-xh6O/CkQoPOWDdYTDqeRdPCVd1SpvCA9XXcUnZS2FmJNp1coAFzvtCN9BmamE+4aHK8yyUHUSCcJHgXloTyT2A==" crossorigin="anonymous" referrerpolicy="no-referrer" />

Now you'll be able to use the icon code from Font Awesome to load icons from their library on your page!

Where to Find and How to Use the Library's Icons

Over 2000 icons are available for free at https://fontawesome.com/icons/

Just enter your search term in the search bar, and then filter by "Free" icons.

After searching, click on the icon you want to use; then in the dialog that pops up, simply copy the code in the HTML tab at the bottom and paste it in your code.

If your icon says "PRO" in the top left corner, it will not load if you attempt to use them. Note that changing to a different style of icon can change it from free to "PRO".

Note that the icons are inline elements, since they're using the <i> element. So they show up side-by-side (or in-line) with the text and elements around them.

And yes, they are choosing to use the <i> element as if "i" stood for "icon" instead of "italics". Not the best practice, but a very common one across the internet.

Since they are inline elements,
you can put icons on either side of a chunk of text
and you can break up the lines using the line break element (<br>).

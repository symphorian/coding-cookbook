The anchor element (also known as a link or hyperlink) can be used to navigate the user to a different location on the current page, or to a different page entirely.

By default, anchor elements are inline elements, meaning they will appear "in line" with other text around them. So if we add three anchor elements right here: External Hyperlinks Internal Hyperlinks Dummy Anchor They will all line up right next to each other, even though if you look in the code, you'll see they're on separate lines.

To use anchor elements, you will need to provide a hyperlink reference, or 'href', attribute. Depending on what you provide to the href attribute, you can create either an external hyperlink or an internal link. Because these concepts can get rather complex, we have split them up into their own Cookbook pages.

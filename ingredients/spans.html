Span elements are the most basic inline element in HTML.

As a reminder, inline elements are place "in line" with other elements on the same line (usually text). Span elements themselves often contain text that we want to apply specific styles to, separate from the other text on the same line.

Similar to div elements, span elements have no styling at all.

The content inside a span element will be grouped inline in the code, but unless you use some kind of styling to distinguish it visually, you probably won't be able to tell on the page.

Adding a Custom Navbar (WordPress)

A video tutorial showing how you can add a custom navbar to your WordPress site

By default, most WordPress Themes will configure the navbar (or menu, using WordPress terminology) to automatically add new Pages to it as you create them.

But what if we want to create a custom menu? Well, we can do that too!

The following video covers:

  • Opening the Customize Menu
  • Opening the Menus Menu in the Customize Menu
  • Creating a new Menu
  • Adding your Home Page (or existing Pages) to the new Menu
  • Creating and adding new Pages to add to the new Menu
  • Adding custom links (that go outside of WordPress) to the new Menu
  • Previewing and saving your changes
  • Editing one of the new Pages we created when we built the new Menu

However, note that this demo only applies to "traditional" WordPress Themes that utilize the Customize Menu.

If you don't see "Customize" under "Appearance", you're probably using one of the "modern" WordPress themes, which we don't recommend.

Now you know how to add and customize a Menu on your WordPress site!

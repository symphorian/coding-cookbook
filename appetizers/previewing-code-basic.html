Previewing Code (Basic)

A tutorial showing how to see the effects of the code you're writing, as you write it!

It is technically possible to just write code and never look at it until you push it up to github.com...but not even the most experienced coders would do that.

Testing is a big part of coding, and in web design, the big test is usually whether what you coded actually looks the way you intended.

There are multiple ways this can be accomplished, but this tutorial is for one of the simplest.

Previewing your Code

Seeing is believing

  1. Previewing HTML code in Visual Studio Code is actually pretty simple, as long as you have things set up properly.

    First, we'll assume you have the HTML file you want to preview open in Visual Studio Code.

    Then, to start, just right-click anywhere inside the HTML file's code.

  2. After right-clicking, select the option that says "Show Preview".

    If you don't see that option, first make sure that you right-clicked inside the HTML code.

    You can only preview an HTML file, with HTML code; nothing else. So if you click in the wrong file or wrong area, you won't get that preview option.

    If you're sure you're right-clicking in the right place, you may not have installed the required Extensions in Visual Studio Code.

    Jump back to the Setup Tutorial Part 1 , and go through the last step again, so that you'll be able to preview your code.

  3. If all is well, in a new panel that popped open to the right, you should see the preview of whatever file you right-clicked on.

    Looks like you're set to go; well done!

    However, depending on the size of your monitor or laptop screen, you may wish that you could view more files at once, or at least change the layout of the different code panels.

    If that resonates with you, check out this other tutorial on how to Reorganize your Workspace!

Now you know how to import a code template into your repository, and even how to preview it! Good luck in your assignment!

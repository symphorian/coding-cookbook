Relabeling the Default Menu "Home" Link (WordPress)

A video tutorial showing how you can alter the labels of custom links in a WordPress menu

Normally, you don't need to worry about relabeling custom links in a WordPress menu, since you label them as you create and add them.

But there is one particular custom link that you get by default that you may want to rename: the "Home" link.

The following video covers:

  • Adding a WordPress "Home" link to your menu
  • Relabeling the link after adding it to your menu

This tutorial is particularly useful for an assignment towards the end of CMS Week, where you create a navbar (menu) in WordPress that matches the navbar in your Portfolio site.

Now you know how to add and relabel a custom link in your WordPress menu!

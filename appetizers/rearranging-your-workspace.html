Rearranging Your Workspace

A tutorial showing how you can rearrange the layout of panels in Visual Studio Code

Developers tend to get used to having a lot of things open at once: applications, browser windows, and yes, Visual Studio Code panels.

Sometimes, we may want to focus on only one file at a time, but a lot of coders like to have many related code files open at once.

We can make that happen in Visual Studio Code, and can even reorganize the layout!

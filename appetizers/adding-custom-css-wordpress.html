Adding Custom CSS (WordPress)

A tutorial showing how you can add custom CSS to your WordPress site

The WordPress Block Editor offers you a lot of customization options, but sometimes, you may find it doesn't let you change things in just the way you want.

If you find that to be the case, you can use a combination of "Additional CSS Class(es)" in the Block Editor and the "Additional CSS" option in the Customize Menu to add your own custom styles to your WordPress site.

The following video covers:

  • Adding a custom CSS class to a Block
  • Opening the Additional CSS Menu in the Customize Menu
  • Writing custom CSS to target our custom CSS class

However, note that this demo only applies to "traditional" WordPress Themes that utilize the Customize Menu.

If you don't see "Customize" under "Appearance", you're probably using one of the "modern" WordPress themes, which we don't recommend.

Now you know how to add custom CSS to your WordPress site!

Adding a Block in the Block Editor (WordPress)

The Building Blocks of WordPress

The Three Methods of Adding a Block

  1. First off, if you haven't already, log in to your WordPress site, and navigate to the Page or Post you want to add a Block to, and enter its Block Editor.

    Or, create an entirely new Page or Post.

    There are three main ways to add Blocks in the Block Editor:

    1. The "slash" method
    2. The "inline [+] button" method
    3. The "header [+] button" method

    Each approach has its own pros and cons, which we'll be going over in the next sections.

The "slash" Method

  1. Let's talk about the first method, which we'll call the "slash" method.

    We're calling it that because, as you can see, whenever you have an empty Paragraph Block in the Block Editor, it's placeholder text says "Type / to choose a block".

  2. If you put your cursor into the placeholder text and type the "/" (slash) character, it will bring up a little popup menu, showing you some of the more common Blocks available to use.

    One of the cons of this approach is that you can only see a handful of Blocks at once.

  3. If you keep typing after the slash, it will customize the results to whatever matches most closely to what you typed (or is related to what you typed).

    For example, if you type "/image", you will see that several image related Blocks show up as options to choose.

    To add a Block from this list, simply click on it.

The "inline [+] button" Method

  1. Let's talk about the next method, which we call the "inline [+] button" method.

    This method is also pretty obviously named: wherever you have the opportunity to add a Block via the "slash" method, you can also add a Block via a square black button with a plus icon [+] inside of it on the same line.

  2. Clicking on this button will bring up a little popup, revealing a small number of Blocks that you can select.

  3. Much like with the "slash" method, you can also search for other Blocks, this time by using the search field at the top of the popup.

    However, unlike the "slash" method, you can get two types of results in the popup:

    1. Block results, that match your search term as closely as possible
    2. Pattern results, related to your search term

    Patterns are pre-built combinations of Blocks used to create a certain layout or design. You can think of them like Block Templates. Feel free to use them! However, they are more complex to use and update.

  4. However, if you don't know what you're looking for, you may not know what Block name to even search for!

    In this scenario, you can click the [Browse All] button, and you'll open up a side panel that contains ALL possible blocks that you can add to the Block Editor.

    This is actually the same panel that we'll see in the last approach to adding a Block, the "header [+] button", so we'll cover that in more detail in the last section.

  5. Before we move on to the last method though, there's something important to know about how you use the inline [+] button inside of more complex Blocks (in particular, Blocks that are containers for other Blocks).

    The Block we'll use as an example here is a Column Block, which can contain any other Block or group of Blocks.

    In this case, if you just click your cursor inside of any existing text inside the Column Block, the [+] button won't show up.

    You have to click on the empty space below or around it to select the container Column Block, and only then will a [+] button show up in the bottom right corner.

    This [+] button will open up the same popup that we saw earlier, but the Block(s) you add will be inserted inside of the Column Block, rather than to the Page or Post itself.

    Alternatively, if you have trouble clicking inside the container, you can also open up the List View (1), and then select the Column Block you want to add a Block into (2), and then the [+] button will show up.

The "header [+] button" Method

  1. Now we come to the last method to add Blocks to a Page or Post: the "header [+] button" method.

    Like it's name implies, this time, the [+] button is found in the header, and it's blue rather than black.

    Clicking on this [+] button will open up a panel on the left, revealing ALL possible Blocks that can be added to the Page or Post.

  2. Luckily, the blocks are grouped into different categories to make it easier identify and find them.

    The categories that appear are slightly dependent on the Theme that you pick, but some common categories are:

    • TEXT
    • MEDIA
    • DESIGN (which encompasses layout related Blocks, like Columns)
    • WIDGETS
    • THEME
    • EMBEDS (if you're curious how to get a YouTube video on your site, this is one way to do it!)

    Selecting a Block from here will add to the Page or Post. Occasionally, you might find that the Block you selected was added in a location you didn't want; in that case, click and drag it in the List View to where you want it to be.

Congrats! Now you know how to add Blocks in the WordPress Block Editor!

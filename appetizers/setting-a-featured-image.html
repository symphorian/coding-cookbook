Setting a Featured Image (WordPress)

A Picture Says a Thousand Words

  1. First off, If you haven't already, log in to your WordPress site, and navigate to the Dashboard.

    If you're on the live version of your site, but the Admin Bar doesn't show up at the top of the screen, you might have been logged out due to inactivity.

    Add /login to the end of the URL, and you'll be able to login again.

    Once in the Dashboard, navigate to the Block Editor of the Post you want to add a featured image to.

  2. Once in the Block Editor, make sure the sidebar on the right is open.

    If it is not, click the little sidebar icon to the right of the Publish button in the navbar.

    Then make sure that you are in the Post tab of the sidebar (not the Block tab).

    Once you confirm you're in the Post tab, scroll down until you find the Featured Image drawer and click on it.

  3. After opening the Featured Image drawer, you should see an empty gray rectangle with just the words "Set featured image".

    Click on that rectangle.

  4. After clicking, a large dialog should appear, prompting you to select or upload image files.

    Alternatively, if you've previously uploaded the image you want to use for the cover image, you can skip this step and go straight to the Media Library tab.

  5. After uploading an image, you will be moved to the Media Library tab of the dialog.

    This tab will show ALL of the images you've uploaded to this particular WordPress site.

    Click on the thumbnail of the image you want to use as a featured image, and then click on the blue "Set featured image" button on the bottom right.

  6. The dialog will close, and you'll find yourself in the Block Editor again.

    But now, you should see the featured image you selected in the Featured Image drawer in the sidebar.

    Note that a featured image will NOT show up in the Block Editor.

    You will only see it where Posts show up on the live version of the site, such as the Home page or the actual page of the Post itself.

    Make sure to Publish your changes before you leave the Block Editor!

  7. If we visit the live version of the site, we'll now see our featured image above the Post that we set it on.

    Note that how a featured image renders and behaves is completely dependent on the Theme used for the site.

    Some Themes will put featured images in different places, and others will ignore them completely.

Congrats! You've successfully set a featured image on a Post!

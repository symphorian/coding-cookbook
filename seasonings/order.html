Additional Examples

NOTE: The order property is a Flexbox Item Property.

This property will have no effect if applied to an element that is not INSIDE an activated Flexbox Container (i.e. has display: flex; applied to it)

The order property can be used to reorder the position of elements inside of a Flexbox Container.

The default order value of all Flexbox Items is 0, and it can be set to any whole number value, whether positive or negative.

Flexbox Items with the order property will arrange themselves relative to the order values of other Items.

No order Properties

We'll start by showing you a basic Flexbox Container, with no order properties set on the Flexbox Items inside.

A
B
C
D

Pretty basic. No reordering occurring.

One order Property

Now, we'll give one Item an order property.

A
B (order: 2;)
C
D

Remember, each Item has a default order property of 0, so by giving an order value of 2 to Item B, it moves to the end, since it has a greater order value than all the other items (2 > 0).

Let's try another, but a negative value this time.

A
B
C
D (order: -1;)

Again, since the default order value is 0, and since we gave Item D an order value of -1, that will move it in front of all the other items (since -1 < 0).

Multiple order Properties

Now, we'll assign multiple order properties.

A (order: 2;)
B (order: 5;)
C (order: 1;)

In this case, the order of the boxes in the HTML are A, B, C; but as you can see, they have been reordered to be C, A, B.

That's expected, since the order property values for C, A, and B are 1, 2, and 5, respectively.

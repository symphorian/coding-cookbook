Additional Examples

The border-radius properties allow us to round the corners of an element.

Despite what the name may imply, you don't technically need to have a border set to use border-radius, as we'll see in some later examples.

Documentation for one value
border-radius: all-sides;

Example
border-radius: 24px;

This div has its border-radius set to 24px on all corners.

When we provide one value to the border-radius property, that value gets applied to all corners of the element's Box Model.

Round Example
border-radius: 50%;
text-align: center;

When we set all corners to have a border-radius of 50%, we can actually get each corner to meet each other in the middle of each edge.

However, the div is still technically a box, and the text will still extend past the rounded borders to the corners of that box if we let it.

We're centering it here just to make it
appear like it's bound by the
rounded corners.

Documentation for two values
border-radius: top-left/bottom-right top-right/bottom-left;

Example
border-radius: 24px 60px;

This div has border-radius applied to all corners, but with different top-left/bottom-right and top-right/bottom-left values.

When we provide two values to the margin property, the first value gets applied to the top-right and bottom-left corners of the Box Model, while the second value is applied to the top-right and bottom-left corners.

Documentation for three values
border-radius: top-left top-right/bottom-left bottom-right;

Example
border-radius: 24px 60px 0px;

This div has border-radius added on all corners, but with different top-left, top-right/bottom-left, and bottom-right values.

When we provide three values to the margin property, the first value gets applied to the top-left corner of the Box Model, the second value is applied to the top-right and bottom-left corners, and the third value is applied to the bottom-right corner.

Documentation for four values
border-radius: top-left top-right bottom-right bottom-left;

Example
border-radius: 60px 8px 24px 100px;

This div has border-radius added on all corners, but with different values for each corner.

When we provide four values to the border-radius property, the values get applied in a clockwise rotation, starting with the top-left corner.

The border-radius property can be added to ANY element, even an image.

It can even be added when an element doesn't have any border properties set.

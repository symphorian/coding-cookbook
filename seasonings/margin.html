Additional Examples

There are several margin related properties and values that we can take advantage of.

The key thing to remember is that margin is applied OUTSIDE an element's border, whether that border is visible or not.

Documentation for one value
margin: all-sides;

Example
margin: 24px;

This div has its margin set to 0px on all sides.

When we provide one value to the margin property, that value gets applied to all sides of the element's Box Model.

Example
margin: 0px;

This div has no margin added on all sides outside of its borders.

As a result, its borders are rammed up right next to the surrounding divs.

Documentation for two values
margin: top/bottom left/right;

Example
margin: 24px 48px;

This div has margin added on all sides, but with different top/bottom and left/right values.

When we provide two values to the margin property, the first value gets applied to the top and bottom of the Box Model, while the second value is applied to the left and right sides.

Documentation for three values
margin: top left/right bottom;

Example
margin: 24px 8px 60px;

This div has margin added on all sides, but with different top, left/right, and bottom values.

When we provide three values to the margin property, the first value gets applied to the top of the Box Model, the second value is applied to the left and right sides, and the third value is applied to the bottom.

Documentation for four values
margin: top right bottom left;

Example
margin: 60px 8px 24px 100px;

This div has margin added on all sides, but with different top, right, bottom, and left margin values.

When we provide four values to the margin property, the values get applied in a clockwise rotation, starting with the top.

Remember that margin is still applied, even if you can't see the border.

This div has the same four margin values as the div above it, even though we removed the border.

Examples of individual margin properties
margin-top: 100px;
margin-right: 24px;
margin-left: 24px;
margin-bottom: 48px;

This div has its margin set by the properties for the individual sides.

These individual properties have a higher priority compared to the generic "margin" property, so if both are used, the individual properties are most likely the values that will be set.

Typically we only use the individual margin properties to tweak one or two sides at a time. If we need to set more, it's usually simpler to use the generic "margin" property from earlier.

Examples of the auto value
margin-right: auto;
margin-left: auto;
width: 250px;

There is a special value that we can use with margin called "auto".

The "auto" value basically says to take up all the available space on the side it's set to.

BUT! It doesn't work on inline, inline-block, or floated elements. And only if an explicit width is set on the element.

We have centered our container!

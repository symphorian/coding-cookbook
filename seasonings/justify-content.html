Additional Examples

NOTE: The justify-content property is a Flexbox Container Property.

This property will have no effect if applied to an element that is not an activated Flexbox Container (i.e. has display: flex; applied to it)

The justify-content property determines how a Flexbox Container both aligns and spaces the content inside of it.

It's important to understand that the justify-content property works along a flexbox's main-axis.

This means that the behavior of this property will change depending on the flex-direction of the Container.

  • if flex-direction is set to row
    • justify-content will work along the x-axis (or horizontal direction)
  • if flex-direction is set to column
    • justify-content will work along the y-axis (or vertical direction)

Basic Code Snippets

background: radial-gradient(red, blue);
background: radial-gradient(at bottom right, yellow, orange, purple);
background: radial-gradient(at 25%, #00000000, #000000FF);
background: radial-gradient(red, white 30%, blue 50%);

Additional Examples

Let's be very clear to start: radial-gradient is NOT a CSS Property!

radial-gradient is a function that can be passed as a value to the background property (well, technically background-image, but we can cheat and use the shorthand property).

If you try to use radial-gradient as its own property, you will get nothing but broken code.

Now, with that out of the way, let's look at some examples.

Two color stops

Gradients are created with values known as "color stops"; color stops essentially tell the gradient what colors to transition between.

One of the simplest color gradients is one between two colors, so let's do that below with the following code:

background: radial-gradient(yellow, green);

Two color stops with position

Above, we created a radial gradient that goes from yellow to green, from the center of the element's background to the edges. What if we wanted that gradient to start in a different position?

In that case, we can specify the position of the gradient start by using the special side keywords of "top", "bottom", "left", and "right", as in the examples below.

background: radial-gradient(at top, yellow, green);

background: radial-gradient(at right, yellow, green);

background: radial-gradient(at bottom left, yellow, green);

We can also directly the position along the x and y axes using percentages, as below.

The first example positions the center of the gradient at 25% of the way from the left in the vertical center. The second example positions the center of the gradient at 75% along the way from the left, and at 100% of the way from the top (i.e. the bottom).

background: radial-gradient(at 25%, yellow, green);

background: radial-gradient(at 75% 100%, yellow, green);

Multiple color stops

Theoretically, there's no limit to the number of color stops you could give to a gradient (though practically, you'd never need THAT many color stops).

Here's an example with multiple color stops:

background: radial-gradient(blue, purple, lightblue, aquamarine, teal);

Color Stops with Length

We already have a lot of flexibility with radial gradient, but what if we wanted even more control?

For instance, what if we wanted to create a gradient that stopped transitioning midway through?

In that case, we could provide a length as a percentage value after a color stop to tell it, well, where that color should "stop" on the gradient. Let's see it in action.

background: radial-gradient(crimson, gold 50%);

See how the gold color transition stops in the middle of the gradient at 50%? That's because of the color stop percentage that we added.

You do have to be a little thoughtful with color stop lengths, or else they may not work properly. As you progress through the gradient, your color stop percentages should always be increasing, since they're referring to points on the gradient.

For instance, it wouldn't make sense to provide color stop lengths in this order: 20%, 80%, 50%.

By the time you've reached the color stop at 80%, you've already passed 50%, and you can't go backwards in a gradient.

Here's another example of color stops with lengths:

background: radial-gradient(red 20%, white, blue 60%);

In this case, red only starts transitioning to white at 20%, and white must transition to blue by 60%, so that limits the transition from red to white to blue between 20% and 60% of the gradient.

Additional Examples

The background-image property is used to add an image to the background of a container.

Below is the image we'll be using for this example. It works well as a background image because there isn't too much detail in it that would distract from any content we put on top of it.

An <img> element

Unlike the <img> element (shown above), an image applied via the background-image property has no size associated with it at all. It's as if we "paint" the image onto the container's background (the same background where background-color would go).

Because of this, you have to be a little thoughtful about how you used this property.

For example, in the <div> container below, we added a background-image ...but you can't really see much of it at all, because the container it was painted onto is very small; it's only the height of the text inside it!

Example 1: A basic background-image property example
background-image: url("../images/abstractBackground.webp");

However, if we stretch out that container a bit with something like the height property, then we can see much more of the image that we painted onto it.

Example 2: A background-image property example with height applied
background-image: url("../images/abstractBackground.webp");
height: 400px;

But, even then, there are still some problems. For instance, if you look closely, the image we painted onto the background is repeating itself over and over (a behavior which is sometimes called "tiling").

To make our background-image behave a bit better, we usually have to bring in a couple of its sibling properties, like background-size and background-position. These properties are frequently found together in CSS code, and work together to address the sizing and positioning issues that are common when you ONLY use background-image.

Example 3: A background-image property example with other common background properties applied
background-image: url("../images/abstractBackground.webp");
background-size: cover;
background-position: center center;
height: 400px;

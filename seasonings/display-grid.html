display: grid;

Additional Examples

Grid is one of the CSS Layout Modules, and the first step to using a CSS Grid is activating a grid container.

To do so is pretty simple: you just assign the display property the value of grid.

Examples of display: grid;

By default, containers are not grid containers, so they use normal Document Flow to position items inside of them.

Block elements have their whole lines and are stacked, and inline elements are arranged in line, same as usual.

display: flex;

However, if we take the same code and activate flexbox on the container, we'll see the elements inside change.

display: grid;

A
B C
D

CSS Grid containers change the normal Document Flow of items inside them, though without additional grid properties set, it can be hard to tell exactly how it changes the Document Flow.

CSS Grids essentially force all elements inside of them into grid cells, so there is no longer any distinction between block and inline items inside of a CSS Grid.

By default, there is only one column in a CSS Grid, which takes up the width of the whole container, and each row grows as large as it needs to be for the content inside of it.

This results in each Grid Item appearing to act as a block element, but this is not actually the case.

Watch what happens when we introduce a new grid property (grid-template-columns) to define additional columns.

grid-template-columns: 1fr 1fr 1fr;

B C
D

Now our four Grid Items are slotted into three columns, and when they fill up a row, they start a new row beneath it, and start filling up the columns in that row.

This is just a brief illustration that CSS Grid containers and items are their own unique beast in CSS. There are lots of ways to change the positions and layouts of Grid items.

To learn more about them, check out the Grid pages in the CSS part of the Coding Cookbook!

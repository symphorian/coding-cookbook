Additional Examples

NOTE: The align-items property is a Flexbox Container Property.

This property will have no effect if applied to an element that is not an activated Flexbox Container (i.e. has display: flex; applied to it)

The align-items property determines how a Flexbox Container both aligns and stretches the content inside of it.

It's important to understand that the align-items property works along a flexbox's cross-axis.

This means that the behavior of this property will change depending on the flex-direction of the Container.

  • if flex-direction is set to row
    • align-items will work along the y-axis (or vertical direction)
  • if flex-direction is set to column
    • align-items will work along the x-axis (or horizontal direction)

The following examples will showcase align-items as it affects a flexbox with a row flex-direction.


align-items: stretch; (default)

This is the default behavior of align-items.

If no explicit heights are set on the Flexbox Items, the Flexbox Container will stretch them out to fill the Container (along the cross-axis).

For this example, we gave a height to the Flexbox Container and removed the heights from the boxes inside, to illustrate the stretch property value.

Basic Code Snippets

vertical-align: baseline;
vertical-align: top;
vertical-align: middle;
vertical-align: bottom;

Additional Examples

The vertical-align property is an easily misused property. As its name implies, it is used to vertically align content...but it only applies to VERY SPECIFIC content. You can't use it on just anything.

Restrictions

Let's be very clear to begin: the vertical-align property can ONLY be used on inline and inline-block elements.

If you try to use it on a block element or anything else, it will do nothing at all.

What's more, unlike the similarly named "text-align" property, the vertical-align property does NOT go on the container of the content you want align vertically.

The vertical-align property is a "self-aligning" property; in other words, you place it on the element whose alignment you want to change.

Examples

Alright, now with the restrictions out of the way, let's see how we can use the vertical align property to modify our inline or inline-block content.

inline-block -> middle

Now let's see what happens when we apply the following code to the inline-block element:

vertical-align: middle;

The "middle" result might be a little surprising, since the inline-block seems to have aligned with the baseline again.

But interestingly enough, the "middle" alignment behavior (according to MDN):

"Aligns the middle of the element with the baseline plus half the x-height of the parent."

In other words, it basically aligns the middle of the element, with the middle of the baseline. So with that in mind, the alignment of the inline-block is actually expected.

all -> top

Finally, let's see what happens when we apply the following code to ALL the elements in the line:

Inline-block

This is probably the most expected result. If we set all elements in a line to have vertical-align set to "top", then they're all going to move to the top of the line.

The vertical-align property is probably the most predictable when EVERYTHING in a line is given the same vertical-align value, so we'll leave the rest of the example to you to try out.

Basic Code Snippets

grid-template-columns: 200px 400px;
grid-template-columns: 1fr 600px 1fr;
grid-template-columns: 20% 30% 30% 20%;

Additional Examples

NOTE: The grid-template-columns property is a Grid Container Property.

This property will have no effect if applied to an element that is not an activated Grid Container (i.e. has display: grid; applied to it)

The grid-template-columns property defines columns to populate a Grid Container.

Grid column widths can be any length value, whether absolute (e.g. pixels) or relative (e.g. percentages); they only need to be separated by spaces in order to set them on the grid-template-columns property.

Examples with absolute widths

Absolute width values are probably the most straightforward to use with a grid, so we'll start an example with those (though they're not the most ideal, as we'll see later.)

Here's an example with only two columns defined.

grid-template-columns: 150px 350px;

Above, you can see the two columns of different widths, and the Items (A, B, C, D, E, and F) populate the rows and columns one by one from left to right, and top to bottom.

This is the normal behavior for all grids, unless some other type of layout property is used (such as grid-template-areas).

Examples with relative widths

Relative width values get a little more tricky, because you have to make sure those values don't surpass 100% of the Container width, but the nice thing about them is they're much more flexible.

grid-template-columns: 20% 60% 20%;

In this example, we have three columns, since we provided three values to grid-template-columns, and this time they are relative values: percentages.

Since we used all percentage values for the columns, that means that the Grid Container and grow or shrink, and the columns will grow and shrink with it.

That would not occur with absolute units. If the Container was too big, there would be empty space left; if the container was too small, then the contents would overflow.

Examples with fractional units

Relative width values are great for making sure columns grow and shrink with a container, but sometimes we need a mix of both rigidity and flexibility.

Enter, the fractional unit!

Fractional units are still relative units, but they have a very interesting behavior (if you've learned about the flex property for Flexbox Items, you'll find they're actually very similar!).

Fractional units are best described as "fluid"; they resize themselves to fill the remaining space left over in a Grid Container, and also relative to any other fractional units used.

Let's look at a example, using fractional units to create flexible left column in a grid layout. This might be a layout where you have your main content on the left, and a static-sized sidebar on the right.

grid-template-columns: 1fr 250px;

When combined with the grid-template-areas property, fractional units can let us create very useful grid layouts with empty gutters on the outside that center our main content and keep our focus on the center of the page.

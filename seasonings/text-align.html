Additional Examples

The text-align property allows us to align content within an element. (Contrary to what the name implies, this property will allow you to align any inline content inside of a container, including elements like images)

Example of text-align: left;

This text is left-aligned (the default alignment).

Example of text-align: center;

This text is center-aligned.

Example of text-align: right;

This text is right-aligned.

It's very important to remember that if you want to use text-align, it must be set on the CONTAINER of the content you want shifted left, center, or right, not on the content itself.

That easy enough to remember with paragraph elements, but what if we have a div element with an img element inside?

On which element do we set text-align on?

Let's try both, and see what happens.

text-align set on the img element

text-align set on the div element

And the winner is...the div element!

And that makes sense, because the text-align property only aligns content WITHIN the element it is set on. It doesn't tell an element how to align itself.

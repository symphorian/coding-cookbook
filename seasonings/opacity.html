Additional Examples

The opacity property is a fun one that affects the transparency of the ENTIRE element that it is applied to.

The opacity property will accept values between 0 and 1.

A value of 1 (or 100%) given to the opacity property will make an element completely opaque, meaning not transparent at all.

A value of 0 (or 0%) given to the opacity property will make an element not opaque at all, completely transparent.

Let's check out how these values can affect content on a page.

This paragraph has black text and is 100% opaque with opacity: 1;

This paragraph may look light grey, but it's actually black.
Because it has opacity: 0.4;, it is 40% opaque (or 60% transparent), so it is actually black mixed with white, which is why it looks light grey.

This paragraph has black text and is 100% opaque with opacity: 1;

This time, the black text is blending in with the pink behind it, since it has opacity: 0.4;.

This paragraph has black text, a yellow background, and a green border, and is 100% opaque with opacity: 1;

This time, the paragraph still has opacity: 0.4; but notice that not only is the text partially transparent, but the background and the border are as well.
In other words, the ENTIRE element is partly transparent.

This paragraph has an image inside it, with the same 100% opacity with opacity: 1;

The paragraph also has an image, still with opacity: 0.4;.

Notice that the image is also partly transparent.

In summary, the opacity property will make an ENTIRE element partially opaque (or partially transparent, depending on how you look at it).

If you apply the opacity property to something and you don't get the result you expect, most likely you're applying opacity to the wrong element.

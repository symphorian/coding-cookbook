display: flex;

Additional Examples

Flexbox is one of the CSS Layout Modules, and the first step to using flexbox is activating a flexbox container.

To do so is pretty simple: you just assign the display property the value of flex.

Examples of display: flex;

By default, containers are not flexbox containers, so they use normal Document Flow to position items inside of them.

Block elements have their whole lines and are stacked, and inline elements are arranged in line, same as usual.

display: block;

However, if we take the same code and activate flexbox on the container, we'll see the elements inside change.

Flexbox containers change the normal Document Flow of items inside them.

They force all elements into a line (horizontal, by default), so there is no longer any distinction between block and inline items inside of a flexbox.

However, flexbox items are not the same as inline items, nor are they the same as block items.

Watch what happens when we apply some heights and widths to our flexbox items.

Normally, inline elements wouldn't change if we applied a height to them, and if we added height to an inline-block, it would grow "up", not "down" like they did in the example above.

This is just a brief illustration that flexbox containers and items are their own unique beast in CSS. There are lots of ways to change the positions, sizes, and alignments of flexbox items.

To learn more about them, check out the Flexbox pages in the CSS part of the Coding Cookbook!

padding: 12px;

Additional Examples

There are several padding related properties and values that we can take advantage of.

The key thing to remember is that padding is applied INSIDE an element's border, whether that border is visible or not.

Documentation for one value
padding: all-sides;

Example
padding: 24px;

This outer div has padding added on all sides, with a border applied to make it easier to see how padding is applied.

When we provide one value to the padding property, that value gets applied to all sides of the element's Box Model.

Example
padding: 0px;

This outer div has no padding on all sides.

As a result, there is no space between its border and this inner div.

Documentation for two values
padding: top/bottom left/right;

Example
padding: 24px 48px;

This outer div has padding added on all sides, but with different top/bottom and left/right values, as well as another border.

When we provide two values to the padding property, the first value gets applied to the top and bottom of the Box Model, while the second value is applied to the left and right sides.

Documentation for three values
padding: top left/right bottom;

Example
padding: 24px 8px 60px;

This outer div has padding added on all sides, but with different top, left/right, and bottom values, as well as another border.

When we provide three values to the padding property, the first value gets applied to the top of the Box Model, the second value is applied to the left and right sides, and the third value is applied to the bottom.

Documentation for four values
padding: top right bottom left;

Example
padding: 60px 8px 24px 100px;

This outer div has padding added on all sides, but with different top, right, bottom, and left padding values, as well as another border.

When we provide four values to the padding property, the values get applied in a clockwise rotation, starting with the top.

Remember that padding is still applied, even if you can't see the border.

This outer div has the same four padding values as the div above it, even though we removed the border.

Examples of individual padding properties
padding-top: 48px;
padding-right: 24px;
padding-left: 24px;
padding-bottom: 100px;

This outer div has its padding set by the properties for the individual sides.

These individual properties have a higher priority compared to the generic "padding" property, so if both are used, the individual properties are most likely the values that will be set.

Typically we only use the individual padding properties to tweak one or two sides at a time. If we need to set more, it's usually simpler to use the generic "padding" property from earlier.

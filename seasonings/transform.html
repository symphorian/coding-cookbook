Basic Code Snippets

transform: rotate(45deg);
transform: scale(1.5);
transform: translateX(150px) rotate(-90deg);

Additional Examples

The transform CSS Property is MASSIVE. There is so much that you can do with it: you can make elements bigger, spin them, and move them...and do all of that together!

An interesting thing about the transform property though is that it doesn't affect document flow at all. It's almost like the transforms we apply are an illusion; they will not change how the rest of the page renders at all (aside from possibly blocking other elements from view).

CSS Function Values

The first thing you need to understand in order to use the transform property is that its acceptable values are called "functions", and we say that the values are expressed in "functional notation".

What does that mean? Well, let's take a look. The following are different examples of function values that can be given to the transform property.

rotate(45deg)
translateY(200px)
scale(1.75)

There are some similarities across the code snippets above: they all have a function name (rotate, translateY, and scale), followed by parentheses, and inside those parentheses are a value that makes sense for the particular function. That is essentially the syntax for any CSS function:

functionName(value)

What functions can be used with the transform property? A LOT!

Next, let's take a look at some of the more common functions used with the transform property, but this is not an exhaustive listing of everything you can do with the transform property.

The rotate() function

As you can probably guess, the rotate() function is used to rotate an element by a specified number of degrees. Let's look at some examples below:

No Rotation
rotate(5deg)
rotate(45deg)
rotate(-90deg)

As you can see, you can use both positive and negative degree values, as long as you specify the units as "deg".

The scale() function

The rotate() function is used to increase or decrease the size of an element by a specified ratio. Let's look at some examples below:

No Scaling
scale(1.5)
scale(0.75)
scale(-0.75)

That last one might be a little hard to read, but it's scale(-0.75).

The way scale values work is this: values greater than 1 will increase the size of the element, while values less than 1 will decrease the size of the element.

To use the examples above, a value of 1.5 will scale the element to 150% of its normal size, while a value of 0.75 will scale the element to 75% of its normal size.

Once again, you can see that we can use both positive and negative values, but negative values have an interesting side effect with the scale function: it will flip the element!

That last example of -0.75 will shrink the element by 75% of its normal size, but also flip it along both the X and Y axes so that it appears upside down.

You can also use functions to only scale along a specific axis (scaleX and scaleY), which will have the effect of stretching or squishing an element, as seen below:

scaleY(0.6)
No Scaling
scaleX(2)
The translate() function

The translate() function is used to move an element along a specific axis (or axes). Let's look at some examples below:

No Translate

translateX(100px)

translateY(100px)

translateX(-50px)
translateY(-50px)

Looks a bit messy, right? Let's see what's happening.

The second box gave translateX() a value of 100px, so it moved the box along the x-axis, 100px to the right of its natural position.

The third box gave translateY() a value of 100px, so it moved the box along the y-axis, 100px down from its natural position.

The fourth box used both translateX() and translateY(), providing a value of -50px to both. That means we first moved it along the x-axis in the negative direction (to the left) by 50px from its natural position. Then we moved it along the y-axis in the negative direction (up) by 50px from its natural position.

As you can see, translating elements can result in them covering or overlapping other elements. They are slightly transparent in the example, so you can see any overlapping elements behind them.

Combining functions

You can actually use the different functions together in a single transform property, and you can get some really crazy effects done that way, particularly once you add animation into the mix.

To use transform functions together, you simply add a space between them when you set the values for the property, as in the examples below:

transform: translateX(50px) rotate(45deg);
transform: scale(0.5) translateY(-50px) rotate(45deg);

However, one important thing to remember is this: the order that you specify the transform functions matters.

Rotating something and then moving it is different from moving something and then rotating it. The reason why gets fairly complex, as it has to do with mathematical concepts like transformation matrices and vector multiplication. Feel free to dig deeper into that if you're a math person!

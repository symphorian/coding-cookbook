Additional Examples

NOTE: The flex property is a Flexbox Item Property.

This property will have no effect if applied to an element that is not INSIDE an activated Flexbox Container (i.e. has display: flex; applied to it)

The flex property can be used to specify the size that a Flexbox Item can grow to, relative to other Flexbox Items that also have the flex property set.

Actually, the flex property is a shorthand property, meaning it allows you to set and configure multiple flexbox-related properties at once; however, we're going to keep our examples here relatively simple.

Check out the Additional Resources section at the end if you want to dive deeper!

NOTE 2: Despite the similarity in name and value, this property is NOT used to activate a Flexbox Container.

Only display: flex; can do that. Don't get them confused!

It can be helpful to think of the flex property as transforming Flexbox Items into a fluid, rather than a solid.

Once you set the flex property on a Flexbox Item, it will grow to fill its container, just like if you poured water into a piece of tupperware.

Mixing flex with width

The flex property gets even more interesting if you mix Flexbox Items with flex with Flexbox Items with a width.

Rememeber, the flex property will size Flexbox Items relative to other Flexbox Items that also have flex properties set.

However, the width property doesn't work like that; it just sizes an element based on the value we provide.

How does that affect things when we mix the two types of Flexbox Items together? Well, let's see.

width: 300px;
1
1
1

In the example above, the size of the first box is fixed at 300px (with a caveat); the other boxes with the flex property, however, don't size themselves relative to it.

The other boxes basically figure out what remaining space is left after the box with the width is sized, and then resize themselves using up the remaining space (again, relative to each other).

The caveat mentioned earlier is that if the Flexbox Container is too small, it will start compressing the items inside of it, even if that means shrinking an item smaller than the explicit width set on it.

Let's look at one other example to wrap up this section.

width: 200px;
1

This is a good example showing the "fluid" nature of the flex property.

The first box has an explicit width of 200px, while the second just has a flex value of 1. But as a result, that second box grows even larger to fill the remaining space.

To use a more realistic analogy, think of the container as a piece of tupperware, and the box with the width of 200px like an ice cube.

Then think of the box with flex: 1; on it like water; as we pour it into the tupperware, it fills all the remaining space inside around the ice cube.

That's a pretty good example of how the flex property works when elements with widths are present.

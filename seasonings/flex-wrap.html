Additional Examples

NOTE: The flex-wrap property is a Flexbox Container Property.

This property will have no effect if applied to an element that is not an activated Flexbox Container (i.e. has display: flex; applied to it)

The flex-wrap property specifies whether content in a flexbox should wrap to a new line if the content inside of it reaches the end of the container.

All flexboxes have a flex-wrap set; by default, they are set to nowrap, but that can be changed to wrap if its what you need.

Example of flex-wrap: nowrap;

Again, nowrap is the default flex-wrap property value. If you don't set flex-wrap yourself, it will automatically be set to nowrap.

In normal HTML containers, if they are not set to wrap, their content will overflow.

However, flexbox containers are different. They will overflow eventually if they have absolutely no space left inside of them, but before they get to that point, they will instead try to compress the size of the items inside.

It is only when they have squeezed all the remaining space from the items inside them that flexbox containers will start to overflow.

Example of flex-wrap: wrap;

If you set the flex-wrap property of a flexbox container to wrap, then it will behave similar to a normal HTML container.

When its contents get too wide, it will simply start a new line inside itself and shift its content onto that line.

Additional Examples

There are many values that can be used with the display property, and some of them are quite complex.

We'll stick to the basics in the following examples.

Examples of display: block;

Setting display: block; transforms an element into a block element, like a <p>, <h1>, or <div> element.

Block elements are pretty straightforward. They take up an entire line in their container, and they stack with other block elements.

They take up an entire line even if the content inside them doesn't need that much space, and even if they don't have any borders or background-color set.

Examples of display: inline;

Setting display: inline; transforms an element into an inline element, like a <span> or <a>.

Inline elements sit next to other inline elements until they reach the end of their container; then they wrap around to the next line.

Simple CSS properties such as font or color can be used to style inline elements with no problems.

That's because inline elements ignore vertical padding, vertical margin, vertical borders... really any vertical part of the Box Model related to spacing.

As a result, if we want to do any sort of styling to an inline element other than simple font or color property changes, we probably want to use the next display value: inline-block.

Examples of display: inline-block;

Setting display: inline-block; gives us the best of both worlds: we can give an element the side-by-side behavior of an inline element, but also have it respect the Box Model like a block element.
